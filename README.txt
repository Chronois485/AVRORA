                                Quick Guide to Using the A.V.R.O.R.A.™ Voice Assistant
A.V.R.O.R.A.™ stands for Artificial Virtual Robot Optimized for Reliable Assistance™.

This guide explains how to use A.V.R.O.R.A.™.

                                        Configuring A.V.R.O.R.A.™
Use the settings.txt file for configuration. Below are examples of settings parameters:

your_name: Enter how you want to be addressed (in English layout, default: sir).
music_link: Insert a link to your favorite music service (default: Spotify).
use_light_system: True or False (controls the lighting system, default: False).
telegram_online: True or False (opens Telegram in a browser or on the computer, default: False).
telegram_path: Specify the full path to Telegram on your computer if not using the browser version.
language: Specify the language (currently available: uk, en).

                                        Using A.V.R.O.R.A.™
To launch A.V.R.O.R.A.™, open the file A.V.R.O.R.A..bat.
To activate A.V.R.O.R.A.™'s listening mode, say "Aurora" before your command.

Command - Action Performed
search [query] - Searches for the requested information in a browser.
goodbye - Turns off the A.V.R.O.R.A.™ voice assistant.
open youtube - Opens YouTube.
open telegram - Opens Telegram (depends on settings).
play music - Opens a link to the music service (depends on settings).
shut down pc - Shuts down your computer.
what's the date - Provides the current date.
microsoft office - Opens Microsoft Office in a browser.
gemini - Opens Google's Gemini AI in a browser.
chat gpt - Opens OpenAI's ChatGPT in a browser.
turn on/off the lights - Controls the lights if configured.
what time is it - Provides the current time.

                                Короткий посібник з використання голосового помічника A.V.R.O.R.A.™
A.V.R.O.R.A.™ розшифровується як Artificial Virtual Robot Optimized for Reliable Assistance™.

У цьому посібнику пояснюється, як користуватися A.V.R.O.R.A.™.

                                        Налаштування A.V.R.O.R.A.™
Використовуйте файл settings.txt для налаштування. Нижче наведено приклади параметрів налаштувань:

your_name: введіть, як до вас звертатися (англійською розкладкою, за замовчуванням: sir).
music_link: вставте посилання на ваш улюблений музичний сервіс (за замовчуванням: Spotify).
use_light_system: True або False (контроль системи освітлення, за замовчуванням: False).
telegram_online: True або False (відкриття Telegram у браузері або на комп’ютері, за замовчуванням: False).
telegram_path: вкажіть повний шлях до Telegram на комп’ютері, якщо використовується не браузерна версія.
language: вкажіть мову (наразі доступно: uk, en).
                                        Використання A.V.R.O.R.A.™
Щоб запустити A.V.R.O.R.A.™, відкрийте файл A.V.R.O.R.A..bat.
Щоб A.V.R.O.R.A.™ почав вас чути, перед командою скажіть "Аврора".
