Короткий посібник з використання голосового помічника A.V.R.O.R.A.™
A.V.R.O.R.A.™ розшифровується як Artificial Virtual Robot Optimized for Reliable Assistance™.

У цьому посібнику пояснюється, як користуватися A.V.R.O.R.A.™.

Налаштування A.V.R.O.R.A.™
Використовуйте файл settings.txt для налаштування. Нижче наведено приклади параметрів налаштувань:

your_name: введіть, як до вас звертатися (англійською розкладкою, за замовчуванням: sir).
music_link: вставте посилання на ваш улюблений музичний сервіс (за замовчуванням: Spotify).
use_light_system: True або False (контроль системи освітлення, за замовчуванням: False).
telegram_online: True або False (відкриття Telegram у браузері або на комп’ютері, за замовчуванням: False).
telegram_path: вкажіть повний шлях до Telegram на комп’ютері, якщо використовується не браузерна версія.
language: вкажіть мову (наразі доступно: uk, en).

Використання A.V.R.O.R.A.™
Щоб запустити A.V.R.O.R.A.™, відкрийте файл A.V.R.O.R.A..bat.
Щоб A.V.R.O.R.A.™ почав вас чути, перед командою скажіть "Аврора".
