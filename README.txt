Короткий посібник по використанню голосового помічника A.V.R.O.R.A.™
A.V.R.O.R.A.™ розшифровується як Artificial Virtual Robot Optimized for Reliable Assistance™.

У цьому посібнику пояснюється, як користуватися A.V.R.O.R.A.™.

        Налаштування A.V.R.O.R.A.™
1. Встановлення Python
Завантажте Python за допомогою файлу python download.exe.
Python — це мова програмування, яка використовується A.V.R.O.R.A.™.
Обов'язково додайте Python до PATH. Щоб зрозуміти, як це зробити, зверніться до нашої підтримки в Telegram: https://t.me/Chronos4.

Запуск setup.bat
Запустіть файл setup.bat, який завантажить усі необхідні бібліотеки для роботи A.V.R.O.R.A.™.

Налаштування голосового помічника
Використовуйте файл settings.txt для налаштування. Нижче наведено приклади параметрів налаштувань:

your_name = введіть, як до вас звертатися (англійською розкладкою, за замовчуванням: sir)
music_link = вставте посилання на ваш улюблений музичний сервіс (за замовчуванням: Spotify)
use_light_system = True або False (контроль системи освітлення, за замовчуванням: False)
telegram_online = True або False (відкриття Telegram у браузері або на комп’ютері, за замовчуванням: False)
telegram_path = вкажіть повний шлях до Telegram на комп’ютері, якщо використовується не браузерна версія

        Використання A.V.R.O.R.A.™
Щоб запустити A.V.R.O.R.A.™, відкрийте файл A.V.R.O.R.A..bat
Щоб A.V.R.O.R.A.™ почав вас чути, перед командою скажіть "Аврора".

Команда	- Що виконує
знайди запит - Знаходить потрібну інформацію в браузері

відкрий youtube - Відкриває YouTube
відкрий telegram - Відкриває Telegram (залежить від налаштувань)

вимкни пк - Вимикає ваш комп’ютер
дата - Повідомляє поточну дату
microsoft office - Відкриває Microsoft Office у браузері
gemini - Відкриває ШІ Gemini від Google у браузері
chat gpt - Відкриває ШІ ChatGPT від OpenAI у браузері
вимкни/увімкни світло - Керує світлом, якщо це налаштовано
котра година - Повідомляє поточний час
